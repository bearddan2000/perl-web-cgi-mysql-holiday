# perl-web-cgi-mysql-holiday

## Description
A small project to demonstrate using
legacy cgi script in a docker container.

## Tech stack
- perl 5

## Docker stack
- nginx:latest
- mariadb:latest

## To run
`sudo ./install.sh -u`
Availble at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
